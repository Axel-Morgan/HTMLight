Вопрос 1

Какая ошибка допущена следующей строке кода: <img url="../images/pict­ure.svg" class="img">

Правильный ответ: url не является атрибутом img, вместо него надо использовать src

  • Нельзя использовать в качестве имени класса имя тега
  • url не является атрибутом img, вместо него надо использовать src
  • Тег img не закрыт
  • В качетсве изображения нельзя использовать файл с расширением svg

Вопрос 2

Если в теге input необходимо сделать переключатель (выбор 1-ого из предложенных вариантов), какое значение атрибута type следует использовать?

Правильный ответ: тег HTML является контейнером, который заключает в себе все содержимое веб-страницы

  • Этот тег не имеет никаких атрибутов
  • Тег HTML является контейнером, который заключает в себе все содержимое веб-страницы
  • Данный тег всегда идет первым
  • Если тег HTML не закрыть, то на сайте ничего не отобразится

Вопрос 7

Зачем нужен тег canvas?

Правильный ответ: позволяет средствами JS создавать окно и рисовать в нем различные объекты

  • Создает окно, в котором пользователь может рисовать (как в Paint)
  • Позволяет средствами JS изменять внешний вид вложенных в этот тег объектов
  • Устаревший тег, предназначенный для показа видео файлов форматов mp4 и ogg
  • Позволяет средствами JS создавать окно и рисовать в нем различные объекты
